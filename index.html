Projects

Washit
Front-end Responsive homepage

Washit is a Uber for dry-cleaning where users can book dry cleaning orders. The orders are assigned to local dry-cleaners based on their location. Dry-cleaners have access to a dashboard and can manage their bookings.


This project was built using Ruby on Rails, HAML, PostGreSQL, JavaScript and jQuery. Authentication with Devise and oAuth, permissions with Cancan.

Citynsider

Citynsider is a map generator based on interests. After selecting a location and a 'tribe' -such as artists, hipsters or hippies- the app will return a path going through the most popular places visited by people belonging to that tribe.


This project was built using Ruby on Rails, Javascript, jQuery, AJAX, Google Map API, Geocoder and Yelp API.

Readr

Readr is a social RSS Feed Reader. Users can add new RSS feeds, subscribe to an existing feed, bookmark, comment and share articles.


This project was built using Ruby on Rails, HAML, CSS and a custom Rake Task to parse RSS and Atom feeds. Extensive model associations, database architecture with PostgreSQL. Group project: use of Github, branches and pair programming.

Muzup

Muzup is a social music sharing website. Users can create profiles, upload songs, videos and pictures that they can share with their friends and comment.


First project. Ruby on Rails, HTML, CSS, model associations and SQLlite. File uplaod using CarrierWave and Amazon S3. Authorisation using Cancan. Search with Ransack.
